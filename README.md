# Infocar

## Start application

```bash
# Clone this project
$ git clone git@github.com:tuliosj/infocar.git

# Open folder
$ cd infocar

# Install dependencies
$ npm install

# Create database table
$ npm run typeorm migration:run

# Run application
$ npm run dev

# Run tests
$ npm run test
```
